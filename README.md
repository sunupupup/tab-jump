# ⚡ Tab Jump - 浏览器标签页快速跳转插件

一个轻量级的 Chrome 扩展，用快捷键解决日常浏览器里两个烦人的问题：

**找 Tab 不用翻** — 浏览器开了几十个标签页，想切到某个页面（比如豆包、Loop）得一个个翻找，或者干脆又开了个新的。现在按 `Ctrl+J` 输入关键词就能直接跳过去，避免重复开标签。

**批量刷新省手动** — 本地连续提交了好几个 PR，GitHub 上那些 PR 页面都要刷新才能看到最新状态，一个个按 F5 太麻烦。现在一键刷新当前窗口所有标签页，全部搞定。

## 🎯 核心功能

### 1. 快捷键快速搜索
- **默认快捷键**：`Ctrl + J`（可在 `chrome://extensions/shortcuts` 自定义）
- 按下快捷键后弹出搜索框，即时搜索匹配结果

### 2. 智能搜索匹配

#### 配置映射优先
- 支持配置关键词到网站的映射（如 `loop → www.loop.com`）
- 配置项优先显示在搜索结果顶部
- 如果配置的网站已在标签页中打开，会显示"已打开"状态

#### 标签页实时搜索
- 自动搜索当前窗口所有已打开的标签页
- URL 匹配时**自动去除参数和 hash**，只匹配 `origin + pathname`
- 使用 `includes` 方式进行模糊匹配

#### 多关键词支持
- 支持空格分隔的多个关键词（如 `loop doc`）
- **所有关键词都必须同时匹配**才能显示在结果中
- 例如：输入 `loop doc` 只会匹配同时包含 `loop` 和 `doc` 的 URL

### 3. 快速跳转
- **已打开的标签页**：直接切换到对应标签页并聚焦窗口
- **配置的网站（未打开）**：在新标签页中打开（自动补全 `https://` 协议）

### 4. 键盘导航
- `↑` / `↓`：上下选择结果项
- `Enter`：跳转到选中的结果
- `Esc`：关闭搜索框

### 5. 配置管理
- 可视化配置页面，支持添加/编辑/删除映射
- 配置自动保存到 `chrome.storage.sync`，支持跨设备同步
- 实时预览和编辑

### 技术栈

- **Manifest V3**：使用最新的 Chrome 扩展 API
- **Chrome APIs**：
  - `chrome.tabs`：查询和管理标签页
  - `chrome.storage.sync`：跨设备配置同步
  - `chrome.commands`：键盘快捷键支持
  - `chrome.windows`：窗口聚焦控制
- **原生 JavaScript**：无依赖，轻量高效
- **CSS3**：现代暗色主题 UI，支持平滑动画

## 📦 安装使用

### 安装步骤

1. **下载或克隆项目**
   ```bash
   git clone <repository-url>
   cd tab-jump
   ```

2. **加载扩展**
   - 打开 Chrome 浏览器，访问 `chrome://extensions/`
   - 开启右上角「开发者模式」
   - 点击「加载已解压的扩展程序」
   - 选择 `tab-jump` 文件夹

3. **配置快捷键（可选）**
   - 访问 `chrome://extensions/shortcuts`
   - 找到 "Tab Jump" 扩展
   - 自定义快捷键组合

### 使用示例

1. **配置映射**
   - 点击扩展图标 → 点击底部「设置」按钮
   - 添加映射：`loop → www.loop.com`
   - 配置自动保存

2. **快速搜索**
   - 按下 `Ctrl + J` 打开搜索框
   - 输入 `loop`，配置的网站优先显示
   - 同时显示当前窗口所有包含 `loop` 的标签页

3. **多关键词搜索**
   - 输入 `loop doc`，只显示同时包含这两个关键词的 URL
   - 例如：`https://www.loop.com/docs/guide` ✅
   - 例如：`https://www.loop.com/home` ❌
